
print() และ echo() ต่างกันยังไง
1.print มีการส่งค่ากลับมา (return) แต่ echo จะไม่มีการส่งค่ากลับไป (void)
2.print สามารถระบุค่าได้ตัวเดียว แต่ echo มีได้หลายตัว
3.echo เร็วกว่า print

ตัวแปร (Variables) ใช้สำรหับเก็บข้อมูลเพื่อใช้ประมวลผล เช่น ตัวเลข ตัวอักษร กลุ่มข้อมูล ข้อความ
โครงสร้างของตัวแปร $ชื่อตัวแปร = ค่าของตัวแปร
กฏการตั้งชื่อตัวแปร
1.สามารเป็นได้ทั้งตัวใหญ่ตัวเล็ก ตัวเลข สัญลักษณ์ _(underscores) และ $(doller sign)
2.อักษรตัวแรกห้ามเป็นตัวเลข
3.ตัวแปร ตัวเล็กตัวใหญ่ มีความหมายต่างกัน
4.ไม่ซ้ำกับ buit-in function (ฟังก์ชั่นมาตรฐานใน PHP)
5.ตัวแปรขึ้นต้นด้วย $(doller sign)
6.ตัวแปรไม่สามารถเว้นวรรคได้ หรือเคาะบรรทัดได้

ชนิดข้อมูลของตัวแปร 
1.Integer ตัวเลขจำนวนเต็ม
2.Float/Double เลขจำนวนจริงหรือทศนิยม
3.Boolean ข้อมูลตรรกศาสตร์มีค่าเป็น True และ Flase
4.String ตัวอักษรหรือกลุ่มข้อความ
5.Array กลุ่มหรือชุดข้อมูลที่มีชนิดเดียวกัน
6.Object การกำหนดให้ตัวแปรนั้นเก็บคุณสมบัติของ Object (Attribute & Method) โดยประกาศใช้งานผ่าน Class (OOP)

ค่าคงที่ (Constant) การสร้างตัวแปรที่เก็บข้อมูลและไม่สามารถเปลี่ยนค่าข้างในได้ (ชื่อตัวแปร,ค่าที่กำหนด) โดยปกติจะใช้เป็นตัวพิมพ์ใหญ่ทั้งหมดเพื่อแยกประเภทของตัวแปรที่สร้างว่าอันไหนเป็นค่าควที่ หรือ ไม่ได้เป็นค่าคงที่
